## About Bunni PC

Bunni PC is a Windows-oriented Roblox script executor made for running Lua scripts through a streamlined desktop app. It targets 64-bit Windows 10 and Windows 11 systems, bringing script execution, a script hub, and development-focused utilities together in a single package.

The emphasis is on quick access to Roblox scripts without a heavy setup process. In addition to Lua execution, Bunni PC offers auto-update support, batch execution mode, a debugger, and persistent queue storage, giving it a practical shape for users who want a simple executor plus script hub workflow.

## FAQ

### Is Bunni PC limited to one Windows release?
It is intended for 64-bit installations of Windows 10 and Windows 11.

### What happens when Roblox updates?
An auto-update engine is included, but actual compatibility still depends on release timing and the current Roblox build.

### How does it compare to other Roblox executors?
Bunni PC combines a lightweight desktop UI, a built-in script hub, batch execution, and debugging tools in one application.

### Can using an executor put my account at risk?
Third-party tools can carry account risk and may fall under Roblox terms. Make sure you understand the rules that apply before running any scripts.

### Does Bunni PC save data on the machine?
Yes, the feature list includes persistent queue storage, so some data may remain locally for convenience.
